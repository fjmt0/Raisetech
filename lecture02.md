以下備忘録
- Gitの初期設定
    - git config --global init.defaultBranch main
    - git config --global user.name "ユーザー名"
    - git config --global user.email "メールアドレス"
- cloneする際はGitからコードをコピーしてくる
- ブランチの作成はgit branch "新しいブランチ名"またはgit switch -c "新しいブランチ名"
- commitはgit commit -m "コメント"で省略可
